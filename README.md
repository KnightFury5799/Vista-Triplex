# Vista Triplex Designer 🏡

An interactive architectural visualization tool for a specialized residential plot in Dhaka, Bangladesh. This project translates a complex set of architectural requirements into a functional, state-driven React application.

## 📍 Project Context
- **Plot Dimensions:** 22 ft x 50 ft (Buildable area)
- **Primary Orientation:** East (Main Entrance) | North (Pond View)
- **Location Logic:** Optimized for tropical cross-ventilation and maximizing the scenic North-facing pond view.

## 🏗️ Architectural Constraints
This project strictly adheres to a specific design brief:
* **Vertical Spine:** A centered 5'x5' square floating staircase connecting all 4 levels.
* **Zoning:** * **Ground:** Public/Guest areas with a hidden service cluster.
    * **1st Floor:** Family-only private wing.
    * **2nd Floor:** Luxury Master Suite + dedicated **PC Workstation/Library**.
* **Environmental Rule:** Strictly zero washrooms on the South side of the building to maintain structural and thermal logic.
* **Rooftop:** 100% open terrace for maximum panoramic utility.

## 🚀 Features
- **Interactive Floor Switching:** Seamlessly toggle between levels to understand the vertical flow.
- **Lifestyle vs. Technical Perspectives:** Visualizes room utility and spatial dimensions simultaneously.
- **Responsive Layout:** Designed to work across mobile and desktop browsers using a CDN-based React architecture.

## 🛠️ Technical Stack
- **Framework:** React 18
- **Styling:** Tailwind CSS (Utility-first CSS)
- **Icons:** Lucide-React
- **Deployment:** GitHub Pages

## 📂 Repository Structure
- `index.html`: Single-entry point containing the React logic and Tailwind configuration.
- `README.md`: Project documentation and architectural brief.
